about

"Kichi ~ Good Luck" is a browser-based chance-operations artwork designed to create a unique, intimate divination-like experience for each viewer using personalized "omens." Each time it is experienced, the system generates a small cluster of fortunes, short poetic lines, and occasional images. Some of these are based on the user's first name and birthday, ensuring the session feels unrepeatable.

The piece opens with a floating Âêâ (the Japanese word for good fortune, "kichi") set against a luminous, Frutiger Aero/early Windows desktop-inspired grass-and-sky field. Viewers enter their first name and birthday at start-up. A sequence of draggable, window-style popups then reveal the personalized content.

how it works

Fortunes are curated by birth month and first initial. Controlled randomness prevents repetition. When a session begins, the system creates a one-time queue with a list of fortunes to display, then shuffles their order. This queue is stored in sessionStorage, ensuring fortunes don't repeat until the page is reloaded.

The system uses three curatorial constraints:

  • one-time month omen ~ the first line of the user's birth-month text appears once at the start, setting a unique, ritual tone
  • no repeats per session ~ each fortune (text or image) appears only once, making each encounter feel specific
  • identity-aware buckets ~ the first letter of a user's name sorts them into one of four buckets (A-F, G-L, M-R, S-Z, plus a default), blending chance and personality

Two people with different names and birthdays receive unique omens. For the same user, shuffled fortunes and pop-up arrangements create new micro-narratives each time.

aesthetic

The aesthetic references early graphical user interface windows and net-art sensibilities ‚Äî blue title bars, dialog boxes labeled "system notice," and compact "ok/yes/cancel" buttons. The floating Japanese character symbolizing good fortune centers the field like a quiet beacon. The popups give the feeling that your computer is getting a virus ‚Äî hopefully a very friendly one.

All windows are click-and-drag, allowing the viewer to curate their layout and create ephemeral constellations. Image fortunes punctuate the text with small, offbeat icons of tenderness and humor.

concept + references

The piece draws on chance (John Cage), Fluxus instructions (Yoko Ono), and early algorithmic graphics (Frieder Nake, Manfred Mohr). It reframes these as a gentle divination machine. Instead of predicting the future, it creates space for attention. Small, ambient prompts nudge noticing.

I wanted randomness to feel gentle, almost intimate ‚Äî not cold or arbitrary. By letting the user's month and initial guide selection and avoiding repetition, the system invites us to see coincidence as a kind of care. Each fortune is short; these are openings for the viewer's personal hopes, worries, or longing.

technical

  • vanilla HTML/CSS/JavaScript ~ no frameworks
  • sessionStorage for queue management
  • shuffle algorithm for controlled randomness
  • draggable windows with z-index management
  • birth month + name initial personalization system

A session lasts 5-10 minutes. The software runs in any modern desktop browser without extra installs.
